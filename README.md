# proyecto-final-estructura-de-datos
Sistema Restaurantes
Aplicación de consola desarrollada en C# (.NET) para gestionar restaurantes, clientes, platos y
pedidos mediante menús interactivos.
- Características principales
1 Crear y gestionar Restaurantes
2 Registrar y listar Clientes
3 Registrar y listar Platos
4 Crear y consultar Pedidos
5 Menú de navegación sencillo por consola
6 Uso de Programación Orientada a Objetos (POO)

- Tecnologías utilizadas
1 C# .NET
2 Programación Orientada a Objetos (POO)
3 Consola / Terminal

- Estructura sugerida del proyecto
/SistemaRestaurantes - Program.cs nnn Models/ - Restaurante.cs - Cliente.cs - Plato.cs - Pedido.cs - README.md

- Cómo ejecutar el proyecto
1. Abrir en Visual Studio / VS Code 2. Ejecutar en consola el comando: dotnet run

- Entidades principales
1 Restaurante: Nombre, Lista de Clientes, Lista de Platos
2 Cliente: Id, Nombre, Teléfono
3 Plato: Id, Nombre, Precio
4 Pedido: Id, Cliente asociado, Lista de Platos, Total a pagar
